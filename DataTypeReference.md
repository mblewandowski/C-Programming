# C Data Types
1. char - a single byte, capable of holding one character in the local character set  
2. int - an integer, typically reflecting the natural size of integers on the host machine. Integers can be any different size as demonstrated below
3. float - single-precision floating point  
4. double - double-precision floating point  

**int:**        0, 100U (Unsigned), 043 (Octal), 0xf453 (Hexadecimal)
**long int:**   3l, 100L, 465345L, 543UL (Unsigned Long)  
**float:**      0.775F, 43.653F, 1e-10  
**double:**     2.12L  
**char:**       'a', '\n', '\0' (Null), '\037' (Octal), '\x0F' (Hexadecimal)
